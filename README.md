## Frontend

In [`frontend/`](./frontend/):

```sh
# Install dependencies
$ yarn

# Build to dist/
$ yarn build

# Start local development server
$ yarn dev
```

## Backend

```sh
# Run central server
$ cargo run --bin server

# Run on computer A5
$ cargo run --bin comp A5
```
